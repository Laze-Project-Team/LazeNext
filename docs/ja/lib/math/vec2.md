# ベクトル2D

2次元のベクトルを表し、2D座標などを表すのに用いられます。

## 初期化

```
実数: x = 0.0;
実数: y = 0.0;
ベクトル2D: ベクトル(x, y);
```

### x

定義するベクトルのx成分を実数で指定します

### y

定義するベクトルのy成分を実数で指定します

## メンバー

### x

ベクトルのx成分

### y

ベクトルのy成分

## メソッド

### 長さ

```
実数:長さ = ベクトル.長さ();
```

#### 戻り値

2Dベクトルの長さ(x²+y²の正の平方根)

### 正規化

```
ベクトル2D: 単位ベクトル = ベクトル.正規化();
```

#### 戻り値

正規化されたベクトル2D(長さ1)
