# モデル.回転()

モデルをある回転軸に対して回転できます。

## 使い方

```
ベクトル3D:軸(1.0, 0.0, 0.0);
実数:角度 = toRad(90.0);
キューブ.回転(軸, ラジアン);
```

### 引数

#### 軸

モデルを回転させる軸[ベクトル3D](/lib/math/vec3)。

#### ラジアン

モデルを回転させる角度を示す実数。度数法の角度を入れないように気を付けること(360度法を[toRad](/lib/math/toRad)を使って変換することもできる)。

### 返り値

なし
