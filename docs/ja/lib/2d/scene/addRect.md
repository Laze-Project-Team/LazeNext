# シーン2D.四角形追加

シーンに四角形(長方形)を追加するためのメソッドです

## 使い方

```
実数: 横幅 = 30;
実数: 縦幅 = 20;
ベクトル2D: 座標(1.0, 2.0);
ベクトル3D: 色(0.2, 0.5, 0.7);
実数: 重なり順 = 10;
実数: id = シーン.四角形追加(横幅, 縦幅, 座標, 色, 重なり順);
```

### 引数

#### 横幅

四角形の横幅を実数で指定します

#### 縦幅

四角形の縦幅を実数で指定します

#### 座標

四角形の座標を[ベクトル2D](/lib/math/vec2)で指定します

#### 色

四角形の色を[ベクトル3D](/lib/math/vec3)で指定します

指定はRGBの順です

#### 重なり順

四角形の重なり順を実数で指定します

重なり順が大きいほど手前に描画されます

### 戻り値

#### id

追加された四角形のidが返ってきます

回転や移動など[スプライト](/lib/2d/sprite/index)に対して処理を行う際に使用します
