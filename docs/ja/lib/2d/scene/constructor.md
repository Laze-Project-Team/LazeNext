# シーン2D.シーン2D

シーン2Dが初期化された時に呼ばれるクラスコンストラクタです

## 使い方

```
実数: 高さ = 100;
ベクトル3D: 背景色(0.2, 0.2, 0.2);
シーン2D: シーン(高さ, 背景色);
```

### 引数

#### 高さ

シーンの高さを実数で指定します

正方形や円などを追加する際の基準になります

横幅はアスペクト比16:9になるように自動で設定されます

#### 背景色

シーンの背景色を[ベクトル3D](/lib/math/vec3)で指定します

指定はRGBの順です
