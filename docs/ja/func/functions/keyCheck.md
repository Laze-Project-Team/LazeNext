# キーチェック

キーが押されているかどうかをチェックします

```
整数: キー
整数: キー入力 = キーチェック(キー);
```

## 引数

### キー

押されているか検知したいキーのキーコードを指定します

## 返り値

### キー入力

キーが押されているかを整数で返します

0であれば押されいないで、1であれば押されています
