# Trigonometric Functions

Trigonometric functions in mathematics.

```
double: PI = 3.141592653589793238;
print(sin(PI)); // 0
print(cos(PI/2)); // 0
print(tan(PI/4)); // 1
```

Note that all arguments are in radians. You can use [To Radian](/lib/math/toRad) to convert degrees to radians.
