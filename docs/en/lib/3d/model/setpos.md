# Model.setPos()

Allows you to set the coordinates of the model.

## Usage

```
Vector3D: coordinates(5.0, 0.0, 0.0);
cube.setPos(coordinates);
```

### Arguments

#### coordinates

The coordinates of the model [Vector3D](/lib/math/vec3).

### Return values

None
