# Model.move()

Allows you to move the model by a certain distance.

## Usage

```
Vector3D: coordinates(5.0, 0.0, 0.0);
cube.move(coordinates);
```

### Arguments

#### Coordinates

The distance the model will move [Vector3D](/lib/math/vec3).

### Return values

None
