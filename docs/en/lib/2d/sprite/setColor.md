# Sprite.setColor

Method to set the sprite's color.

## Usage

```
Vector3D: color(0.6, 0.7, 0.8);
Sprite.setColor(color);
```

### Arguments

#### color

Sets the sprite's color in vector 3D.

The colors are specified in RGB order.

### Return values

None
