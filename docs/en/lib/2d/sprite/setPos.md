# Sprite.setCoordinates

Method used to set the coordinates of the sprite.

## Usage

```
Vector2D: coordinates(2.0, 5.0);
sprite.setCoordinates(coordinates);
```

### Arguments

#### coordinates

Coordinates to set in [Vector2D](/lib/math/vec2).

### Return values

None
