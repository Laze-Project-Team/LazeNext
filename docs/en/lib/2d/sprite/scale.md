# Sprites.scale

Method for scaling a sprite.

## Usage

```
double: factorX = 2;
double: factorY = 2;
sprite.scale(factorX, factorY);
```

### Arguments

#### factorX

Scaling factor in the X direction as a real number.

#### factorY

Scaling factor in Y direction as a real number.

### Return values

None
